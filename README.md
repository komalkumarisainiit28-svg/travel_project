🌍 Travel Website

Welcome to my Travel Website! This project is a modern and responsive travel platform created using HTML and CSS. It provides users with an attractive interface to explore travel destinations, register, and start their journey with us.
✨ Features

* 🏠 Beautiful and responsive Home Page
* 🌎 Explore travel destinations
* 🧭 Easy navigation bar
* 🔐 Login page
* 📝 User registration page
* 🎉 Thank you page after successful registration
* 🔗 LinkedIn and GitHub profile links
* 📱 Mobile-friendly design
* 🎨 Modern UI with a professional color theme

🛠️ Technologies Used

* HTML5
* CSS3
